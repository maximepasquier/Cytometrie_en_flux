# Cytometrie_en_flux

Le projet constitue une première approche de visualisation de données cytométriques en utilisant C++ et le framework Qt. L’objectif est de représenter les données en ne sélectionnant que les marqueurs pertinents et de réaliser différentes segmentations au sein des données. Le principal défi résidait dans les méthodes d’échantillonnage utilisées par les bibliothèques logicielles. Ce travail propose une solution basée sur Qt et QCustomPlot, ouvrant ainsi la voie à des analyses plus approfondies et à l’interprétation des données.
